📋 Google API Setup Guide

Follow these steps to get your free Google OAuth Client ID:

  1. Go to console.cloud.google.com
  2. Click New Project → name it "AutoParts Shop" → Create
  3. Go to APIs & Services → Library
    Enable: Google Sheets API and Google Drive API
  4. Go to APIs & Services → OAuth consent screen
    Choose External → fill in App name → Save
  5. Go to APIs & Services → Credentials
    Click + Create Credentials → OAuth Client ID
  6. Application type: Web application
    Authorised JavaScript origins — add:
    http://localhost   file://
  7. Copy the Client ID and paste it into the app

This is 100% free. Google will never charge you for this use.
